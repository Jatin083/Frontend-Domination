⭐ Timeline: ⭐
var tl = gsap.timeline();  Jab ek animation khatam ho, doosra shuru hoga bina delay ke


✨ Timeline Example ✨

Example:-

var tl = gsap.timeline();

tl.to("#box1", {
    x: 1200,
    delay: 1,
    duration: 2,
    rotate: 360,
    scale: 0.5,
});

tl.to("#box2", {
    x: 1200,
    duration: 2,
    rotate: 360,
    scale: 0.5,
});

tl.to("#box3", {
    x: 1200,
    duration: 2,
    rotate: 360,
    scale: 0.5,
});
